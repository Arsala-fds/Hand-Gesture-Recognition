# 🎨 AI Drawing App (Hand Gesture Based)

This is a Computer Vision project built using OpenCV and MediaPipe.

## 🚀 Features

- Draw using index finger
- Select colors using gesture
- Black color supported
- Clear canvas
- Save drawing as image
- Mode indicator

## 🖥️ Tech Used

- Python
- OpenCV
- MediaPipe
- NumPy

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python main.py

Press Q to exit.
